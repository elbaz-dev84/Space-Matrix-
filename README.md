# Space Dashboard | Matrix84

لوحة تحكم فضائية تفاعلية لمهمة Artemis II مع دعم ثنائي اللغة (عربي/إنجليزي).

## 🌟 المميزات

- **العداد التنازلي** لإطلاق Artemis II مع مزامنة تلقائية من API
- **نظام تعليقات** كامل مع إمكانية التعديل والحذف
- **دعم ثنائي اللغة** (العربية والإنجليزية) مع حفظ الإعدادات
- **تصميم فضائي** مع نجوم متحركة وتأثيرات زجاجية
- **محتوى غني** عن أسرار الكون ومهام الفضاء
- **متجاوب** يعمل على جميع الأجهزة

## 🚀 التقنيات المستخدمة

- **HTML5** - هيكل الصفحة وSemantic Elements
- **CSS3** - تصميم متجاوب مع Custom Properties وAnimations
- **JavaScript ES6+** - وظائف تفاعلية وAPI Integration
- **LocalStorage** - حفظ البيانات محلياً
- **Google Fonts** - خطوط Tajawal وInter

## 🔧 التثبيت والتشغيل

1. قم بتنزيل الملفات أو استنساخ المشروع
2. افتح `index.html` في متصفح الويب
3. الموقع يعمل محلياً بدون حاجة لخادم

## 🌐 اللغات المدعومة

- **العربية (ar)** - اللغة الافتراضية
- **الإنجليزية (en)** - يمكن التبديل عبر الزر في أعلى الصفحة

## 📡 API المستخدم

- **The Space Devs API** - للحصول على مواعيد الإطلاقات الفضائية
- مع fallback محلي في حالة فشل الاتصال

## 💾 البيانات المحفوظة

- تفضيل اللغة (`spaceDashboardLang`)
- تاريخ الإطلاق المحلي (`spaceDashboardLaunchDate`)
- التعليقات (`spaceMatrixComments`)
